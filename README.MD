# To-Do List Web App (Flask)

## 📌 Description

A simple web-based To-Do List application built using Python Flask. Users can add and manage tasks dynamically.

## 🚀 Features

* Add tasks
* Delete tasks
* Simple UI

## 🛠️ Tech Stack

* Python
* Flask
* HTML
* CSS

## ▶️ Run

pip install flask
python app.py
